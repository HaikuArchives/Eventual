
Eventual
=================
Personal time management system for Haiku. Intended to be a full-featured organizer.

![Screenshot](Screenshot.png)

Features:

•Event name<br>
•Event location<br>
•Add event category<br>
•Event time system<br>
•Reminder<br>
•Sound signal<br>
•Note<br>
